# Symbolic-reasoning_meth
定位：
    象征推理
        将模糊的象征符号组合并拆解为规则，生成结构化输出（不掩饰不确定性）
    是什么：
    ◦ 可编程的象征推理模式
    ◦ 塔罗为第一个跑通案例
    不是什么：
    ◦ 不是玄学产品
    ◦ 不是通用AI（AI在其中扮演语言缝合作用）
    
    区别：
    ◦ 较透明（规则清楚）
    ◦ 可迁移：换字典、增添规则 ➡ 换领域（塔罗已验证，紫微斗数正在尝试验证，其他待探索）
    ◦ 多粒度（简易/详细/探究）
    
    适用于：
    ◦ 塔罗解读（已验证）
    ◦ 另类“镜子”
    ◦ 任何有可换为“象征物”的领域（理论可行）
    不适用于：
    ◦ 精确要求高、确定性任务等

• 备注：核心在于模糊与可定义之间交界点化为结构化输出
